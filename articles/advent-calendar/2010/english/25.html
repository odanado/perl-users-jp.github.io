FrePAN - Fresh CPAN Site

tag

FrePAN is one of the recent alternatives to The CPAN Search Site. FrePAN is written and maintained by tokuhirom, and its source is available at github. It's fairly easy to install (FrePAN is based on the Plack/PSGI through his web application framework, Amon2), and it uses really fresh information about who uploaded what, so you usually can view sources and meta information within a few minutes. If you are keen to know the latest information, do visit there, and also try cpanf, which is written by miyagawa. With that, you can query and install the fresh uploads from the shell. If you can wait a bit longer, cpanm would be a fine alternative, which relies on an hourly mirror.

Wish you happy holidays!
